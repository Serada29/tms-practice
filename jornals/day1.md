# Local storage

- stores data inside the browser
- even when the browser refresh the data is still there
- only stores strings

syntax :
localStorage.setItem("name", "jessie")
localStorage.getItem("name")

# Serialization

- converting objects to string

syntax : JSON.stringify()

# Deserialization

- converting string back to object

syntax : JSON.parse()

# FormData

- reads the form and collects all inputs with a name attributes

syntax : new FormData(form)

# Object.fromEntries()

- converts key values pair into an object

# .trim()

    - string method
    - removes whitespaces from the beginning and the end of a string.

# Object.key()

    - gets all property names(key) from an object

# for...in → Iterates over keys / property names

- for...in loops through the keys of an object.

# for...of → Iterates over values

- for...of loops through values inside iterable objects.
